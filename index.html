Modeling Processes of Neighborhood Change (MPONC)

This research project simulates the impact of the Atlanta Beltline on neighborhood gentrification using game theory and no-regret dynamics. The simulation models agent movement across census tracts, with agents seeking to minimize costs based on factors like amenity density. The simulation outputs include dynamic visualizations and CSV data tracking population and income changes across census tracts, providing insights into urban development patterns.

  • Python
  • NumPy
  • Pandas
  • Matplotlib
  • Git
